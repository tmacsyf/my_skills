# 股票分析统一调度

## 这是什么

本 Skill 是**调度中心**，不是分析引擎。它根据你的问题类型，自动判断该走哪条分析路线，然后加载对应的专业 Skill。

## 四类场景自动识别

### 场景一：战略研判 → 缠论主攻
**关键词**："处于什么阶段""什么结构""缠论""中枢""背驰""买卖点信号""大盘结构""周线"

**路由**：`chanlun-trading-system`（主力）+ `stock-technical-analysis`（辅助验证）

**回答什么**：当前处于趋势/盘整/转折？有没有有效的买卖点信号？中枢边界在哪？

### 场景二：战术分析 → 四步分析法主攻
**关键词**："今天怎么样""复盘""该买还是该卖""被套""支撑压力""MACD""均线""仓位"

**路由**：`stock-technical-analysis`（主力）+ `eight-maxims-trading`（辅助执行建议）

**回答什么**：趋势方向、买卖点价位、支撑压力、仓位计划、风险收益比

### 场景三：盘中执行 → 八句口诀主攻
**关键词**："早盘""尾盘""急跌""急拉""涨停""T+0""跳水""追板""做T"

**路由**：`eight-maxims-trading`（主力）+ `stock-technical-analysis`（辅助趋势判断）

**回答什么**：现在该不该动手？什么时机操作？T+0怎么T？

### 场景四：完整复盘 → 三层全走
**关键词**："全面复盘""完整分析""持仓诊断""三合一"

**路由**：`chanlun-trading-system` → `stock-technical-analysis` → `eight-maxims-trading`（逐层递进）

**回答什么**：从大结构到具体操作，给出完整的分析链路

## 输出规范

每次分析开头必须标注：

```
> 本次使用：**{主力技能}({层级}) + {辅助技能}({层级})**
> 场景：{场景名}
```

## 降级规则

| 条件 | 动作 |
|------|------|
| 缠论结构不清/未通过 | 后续结论降级为"观察"，不给"买入确认" |
| 四步分析法趋势偏空 | 八句口诀只做减法（减仓/止损），不做加法（抄底/加仓） |
| 大盘跌破60日线且无主线 | 全局降级为空仓观望模式 |
| 用户明确指定用某个skill | 以用户指定为准，但仍标注在统一体系中的位置 |

## 使用方式

**自动模式**（推荐）：直接问你的股票问题，我会自动识别场景并加载对应Skill。

**手动模式**：你也可以直接指定，如"用缠论分析XX""按八句口诀看XX"。
