# 股票技术分析实战指南

## 概述
本 Skill 提供完整的股票技术分析知识体系与标准化分析工作流，用于指导 AI Agent 对个股进行专业的技术面分析，输出结构化的判断与操作建议。

核心分析方法遵循「四步分析法」：定大趋势 → 找买卖点 → 验真假 → 定计划。

## 标准技术分析工作流（必遵循）

### 第一步：定大趋势（均线系统）
**操作要点**：
1. 看60日均线，判断牛熊
   - 股价在60日线上方，且60日线向上 → 牛市，积极做多
   - 股价在60日线下方，且60日线向下 → 熊市，谨慎观望
2. 看均线排列
   - 多头排列（5日>10日>20日>60日）→ 强势，持有
   - 空头排列（5日<10日<20日<60日）→ 弱势，观望
   - 均线缠绕 → 震荡市，高抛低吸
3. 看股价与均线关系
   - 股价沿5日线上涨 → 超强
   - 股价在10日和20日之间 → 正常
   - 股价跌破20日线 → 警惕

**输出结论**：当前是牛市/熊市/震荡市，大趋势方向是上涨/下跌/横盘

---

### 第二步：找买卖点（MACD + KDJ）
**操作要点**：
1. 看MACD
   - 零上还是零下 → 判断多空市场
   - 金叉还是死叉 → 买卖信号
   - 红绿柱变化 → 动能变化
   - 是否有背离 → 趋势是否衰竭
2. 看KDJ
   - 超买区（80以上）→ 注意回调
   - 超卖区（20以下）→ 关注反弹
   - 金叉死叉 → 短线买卖点
   - 是否钝化 → 单边行情不用KDJ
3. 多指标共振
   - MACD金叉 + KDJ低位金叉 → 强烈买入信号
   - MACD死叉 + KDJ高位死叉 → 强烈卖出信号

**输出结论**：当前处于什么位置，最佳买点/卖点在哪里

---

### 第三步：验真假（量价关系）
**操作要点**：
1. 看量价配合
   - 价涨量增 → 健康上涨，可持续
   - 价涨量缩 → 上涨乏力，警惕
   - 价跌量增 → 放量下跌，危险
   - 价跌量缩 → 下跌动能减弱，可能企稳
2. 看成交量级别
   - 放量/缩量/天量/地量
   - 换手率高低
   - 量比大小
3. 看主力行为
   - 建仓/洗盘/拉升/出货
   - 结合位置判断

**输出结论**：当前走势是真的还是假的，资金是否支持

---

### 第四步：定计划（仓位与风控）
**操作要点**：
1. 确定关键价位
   - 支撑位：第一支撑、第二支撑
   - 压力位：第一压力、第二压力
2. 制定操作策略
   - 买入策略：什么价位买，买多少，分几批
   - 卖出策略：什么价位卖，卖多少，分几批
   - 止损策略：止损位在哪里，跌破怎么办
3. 仓位管理
   - 总仓位控制
   - 单只个股仓位
   - 分批建仓/减仓计划
4. 风险收益比
   - 计算潜在收益和潜在风险
   - 风险收益比至少1:2才值得参与

**输出结论**：完整的交易计划，包含买卖点、仓位、止损止盈

## 常见场景快速响应模板

### 场景一：个股分析请求
**用户问**："帮我看看XX股票怎么样？"
**输出结构**：
- 基本信息（当前价、涨跌幅等）
- 趋势判断（均线系统）
- 技术指标分析（MACD + KDJ）
- 量价分析
- 支撑位与压力位
- 操作建议（分情景）

### 场景二：指标信号咨询
**用户问**："XX股票MACD金叉了，可以买吗？"
**回答逻辑**：
- 先给结论：可以买/谨慎买/不建议买
- 再说理由：从位置、趋势、量能等角度分析
- 最后给建议：仓位、止损位、目标位

### 场景三：被套怎么办
**用户问**："我XX股票被套了，怎么办？"
**分析要点**：
1. 趋势判断：大趋势是涨还是跌
2. 套牢幅度：浅套（<10%）/中套（10%-30%）/深套（>30%）
3. 位置判断：高位还是低位
**回答逻辑**：
- 先分析当前走势和位置
- 根据套牢程度给出不同建议
- 浅套：反弹减仓或止损
- 中套：波段操作降低成本
- 深套：不建议割肉，但也不要补仓
- 强调纪律：设好止损，不要死扛

### 场景四：交易计划制定
**用户问**："我想买XX股票，帮我制定个交易计划"
**输出内容**：
- 标的名称和代码
- 买入理由（技术面/基本面）
- 入场价位和时机
- 仓位比例和建仓批次
- 止损位和止损策略
- 止盈位和止盈策略
- 风险收益比
- 注意事项

## 核心原则
1. **概率游戏**：技术分析是概率游戏，没有100%准确，追求高胜率即可
2. **多指标共振**：单个指标信号不可靠，多个指标同时发出信号才可信
3. **大周期管小周期**：周线级别 > 日线级别 > 60分钟级别，小周期服从大周期
4. **量在价先**：成交量比价格更真实，量价配合的走势才可靠
5. **纪律大于预测**：预测错了很正常，关键是错了能及时止损，控制损失

## 在统一分析体系中的位置（三层体系）

本 Skill 是**战术层**，回答："当前该买还是该卖？什么价位？什么仓位？"

| 角色 | 说明 |
|------|------|
| 定位 | 战术层 —— 多指标共振验证（均线+MACD+KDJ+量价），制定具体交易计划 |
| 前置依赖 | 建议先用 `chanlun-trading-system`（缠论）定大结构方向，避免逆势操作 |
| 适用场景 | 每日复盘、买卖点判断、支撑压力位计算、仓位管理、被套应对 |
| 盘中衔接 | 交易计划制定后 → 盘中执行交给 `eight-maxims-trading`（八句口诀） |
| 独立使用 | 也可以独立使用，但如果缠论显示结构风险，本层结论应降级为谨慎/观察 |

> 在大盘中：四步分析法的买卖点结论，需要缠论的结构背书才升级为"确认"。
