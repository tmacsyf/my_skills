# Finance Skills

A股股票技术分析技能集，三层递进分析体系。

## 技能列表

### stock-unified-analysis
统一调度入口。根据用户问题自动识别场景，路由到战术/执行层的专业技能。

### stock-technical-analysis
四步分析法：定大趋势（均线）→ 找买卖点（MACD/KDJ）→ 验真假（量价）→ 定计划（仓位风控）。

### eight-maxims-trading
八句口诀实战战法：早盘急跌T+0、尾盘异动应对、涨停板策略、20日生命线、空仓纪律、筹码峰识别。

## 依赖

- `chanlun-trading-system`（缠论）：需从 RedSkill 商店单独安装，作为战略层配合使用

## 快速开始

将所需技能目录复制到你的 Agent 技能的 skills 目录中即可。
